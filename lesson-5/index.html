Свойство length – длина, или, иными словами, количество элементов в массиве. Значение свойства length всегда на еди­ницу больше, чем самый высокий индекс массива. Метод для создания массива путем объединения нескольких массивов. Результат получается объединением массива, из которого вызывается метод с массивом или значениями, переданными аргументами методу. -->
